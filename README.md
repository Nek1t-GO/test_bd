# test_bd
Приложение по работе с базами данных с шифрованием паролей

## Генератор хэшей паролей

В проекте реализован генератор хэшей паролей для подключения к базам данных PostgreSQL. Он находится в файле `lib/utils/password_encryptor.dart`.

### Реализация

Класс `PasswordEncryptor` предоставляет два статических метода для генерации хэшей:

1. **MD5-хэш** (`md5PasswordHash`):
   - Генерирует MD5-хэш в формате, совместимом с PostgreSQL: `'md5' + md5(password + username)`.
   - Использует библиотеку `crypto` для вычисления MD5.

2. **SCRAM-SHA-256 хэш** (`scramSha256PasswordHash`):
   - Предоставляет упрощенную реализацию SCRAM-SHA-256 для демонстрационных целей.
   - Формат: `SCRAM-SHA-256\$iterations:saltBase64\$hashBase64:simplified`.

### Тестирование

Тесты для генератора хэшей находятся в файле `test/password_encryptor_test.dart`. Они проверяют корректность формата генерируемых хэшей.
